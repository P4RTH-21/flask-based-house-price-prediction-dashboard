# California House Price Predictor

A Flask-based Machine Learning web application that predicts California house prices using a trained Random Forest model.

## Features

- Flask Backend
- Machine Learning Prediction
- Data Preprocessing Pipeline
- Responsive Dashboard UI
- Real-time House Price Prediction

## Technologies Used

- Python
- Flask
- Pandas
- Scikit-Learn
- Joblib
- HTML
- CSS
- Jinja2

## Project Structure

```text
HousePricePredictor/
│
├── app.py
├── model.pkl
├── pipeline.pkl
├── requirements.txt
│
├── templates/
│   └── index.html
│
└── static/
    └── style.css
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
